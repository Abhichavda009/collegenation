# Test Checklist

Use this checklist before publishing the theme.

## Global

- [ ] Theme uploads successfully.
- [ ] `shopify theme check` has no blocking errors.
- [ ] Header appears on all public pages.
- [ ] Footer appears on all public pages.
- [ ] Main navigation links work.
- [ ] Footer links work.
- [ ] Search opens and submits correctly.
- [ ] Cart icon count updates.
- [ ] No console errors on page load.
- [ ] No missing image icons.
- [ ] No broken asset requests.

## Homepage

- [ ] Hero slider renders.
- [ ] Hero first image loads quickly.
- [ ] Hero links work.
- [ ] Featured products render from Shopify products/collections.
- [ ] Promo tiles render.
- [ ] Shop by collection tabs work.
- [ ] University collection cards render.
- [ ] Lifestyle banners render.
- [ ] Trending products render.
- [ ] Homepage is usable on mobile.

## Navigation

- [ ] Main menu uses Shopify menu links.
- [ ] University selector opens and closes.
- [ ] University tabs switch correctly.
- [ ] Account link routes correctly.
- [ ] Cart link opens page/drawer according to theme settings.
- [ ] Keyboard focus is visible.

## Products

- [ ] Product title, vendor, SKU, price, and description render.
- [ ] Product gallery renders.
- [ ] Thumbnail switching works.
- [ ] Variant buttons update selected variant.
- [ ] Quantity increase/decrease works.
- [ ] Add to cart works.
- [ ] Buy Now button works.
- [ ] Sold-out state works.
- [ ] Specifications metafield renders when present.
- [ ] Product recommendations render.
- [ ] Product structured data validates.
